## 7. รายละเอียดตรรกะการทำงาน (Logic Details)

### 7.4 ตรรกะการแจ้งเตือนระบบ Visit (Library Visit Logic) **[NEW]**
1. **Auto-close Notification**: เมื่อระบบ Janitor ทำการปิด Session ให้อัตโนมัติ (เนื่องจากเลยเวลาปิดห้องสมุด) ระบบจะส่งแจ้งเตือน: *"ระบบได้ทำการเช็คเอาท์ให้คุณอัตโนมัติเนื่องจากห้องสมุดปิดทำการ ขอบคุณที่ใช้บริการ"*
2. **Stay-late Warning**: (Option ในอนาคต) หากสมาชิกอยู่ในห้องสมุดนานเกิน 4 ชั่วโมง ระบบอาจส่งแจ้งเตือนสอบถามความสบายดี หรือเตือนให้พักสายตา

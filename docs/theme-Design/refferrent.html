หลักการออกแบบ Native Web App

1. Responsive First

  • Mobile: 1 คอลัมน์ + เมนูบน
  • Tablet: Icon Sidebar + 2 คอลัมน์
  • Desktop: Full Sidebar + 3 คอลัมน์

2. Touch Target

  • ปุ่มอย่างน้อย 44×44 (iOS) / 48×48 (Android)
  • ระยะห่างระหว่างปุ่มอย่างน้อย 8px
  • ใช้ btn-press เพื่อแสดงการกด

3. Safe Area

  • ใช้ env(safe-area-inset-...)
  • Header: เว้นพื้นที่ด้านบน + notch
  • Bottom Nav: เว้นพื้นที่ด้านล่าง + home indicator
